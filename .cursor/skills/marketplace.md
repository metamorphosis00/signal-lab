# Marketplace Skills

Подключённые skills из маркетплейса Cursor для проекта Signal Lab.

## 1. ESLint
**Зачем:** Автоматически находит ошибки в TypeScript/JavaScript коде.
В Signal Lab следит за соблюдением conventions из rules/conventions.md.

## 2. Prettier
**Зачем:** Автоматически форматирует код единым стилем.
Гарантирует что весь код в backend и frontend выглядит одинаково.

## 3. Prisma
**Зачем:** Подсветка синтаксиса и автодополнение для schema.prisma.
Помогает писать модели и миграции без ошибок.

## 4. Docker
**Зачем:** Подсветка синтаксиса для docker-compose.yml и Dockerfile.
Автодополнение для команд и параметров Docker.

## 5. Tailwind CSS IntelliSense
**Зачем:** Автодополнение классов Tailwind прямо в JSX/TSX файлах.
