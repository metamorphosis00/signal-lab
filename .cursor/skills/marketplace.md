# Marketplace Skills

Подключённые skills из маркетплейса Cursor для проекта Signal Lab.

## 1. ESLint
**Зачем:** Автоматически находит ошибки в TypeScript/JavaScript коде.
В Signal Lab следит за соблюдением conventions из rules/conventions.md.

## 2. Prettier
**Зачем:** Автоматически форматирует код единым стилем.
Гарантирует что весь код в backend и frontend выглядит одинаково.

## 3. Prisma
**Зачем:** Подсветка синтаксиса и автодополнение для schema.prisma.
Помогает писать модели и миграции без ошибок.
Закрывает: работу с Prisma schema, которую custom skills описывают концептуально.

## 4. Docker
**Зачем:** Подсветка синтаксиса для docker-compose.yml и Dockerfile.
Автодополнение для команд и параметров Docker.
Закрывает: работу с infra файлами которых нет в custom skills.

## 5. Tailwind CSS IntelliSense
**Зачем:** Автодополнение классов Tailwind прямо в JSX/TSX файлах.
Показывает какой CSS применится при наведении на класс.
Закрывает: frontend styling, которую custom skills не покрывают детально.

## 6. GitLens
**Зачем:** Показывает кто и когда изменил каждую строку кода.
Помогает понять историю изменений в проекте.

## 7. Thunder Client
**Зачем:** REST клиент прямо в редакторе.
Удобно тестировать endpoints /api/scenarios/run и /api/scenarios/history без curl.
Закрывает: быстрое тестирование API без переключения между приложениями.

## 8. Error Lens
**Зачем:** Показывает ошибки TypeScript прямо на строке кода.
Не нужно смотреть в панель Problems — ошибка видна сразу.

## Почему этот набор

Custom skills в `.cursor/skills/` покрывают концептуальные паттерны (как добавить метрику, как структурировать сценарий). Marketplace skills закрывают то что custom skills не покрывают — инструментальную поддержку: форматирование, подсветку, тестирование API, git history. Вместе они дают полную среду разработки без ручной настройки.

## Как подключить
1. Открой Cursor
2. Нажми Ctrl+Shift+X (Extensions)
3. Найди каждый плагин по названию
4. Нажми Install
